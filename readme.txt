За допомогою даної інформаційно сторінки, користувач може дізнатись наступне:
1. Геолокацію, яку виведено в окремий блок, та за допомогою google.maps показано саме розташування;
2. Погодні умови, а саме температуру в С, загальну характеристику часу доби (холодно/не холодно);
3. Часову зону, ку також відображає блок погоди.
4. Курс місцевої валюти відносно EUR;
5. Місцеві новини ( в даному випадку це новини спорту), також є можливість через посилання перейти на офіційну сторінку новин;
6. Перейти за посиланням на деєкі соцмережі (розташовані в "підвалі" сторінки);

Для отримання даної інформації використовувались сторонні ресурси. За допомогою API була можливість доступитись до потрібних
даних.
Для обробки API запитів використовував метод fetch(). 
У роботі сторінки інколи бувають несправності, які призводять до виводу новин зовсім не місцевих.
Це пов'язано з тим, що більшість ресурсів, які надають можливість розробникам працювати з API є платними.
Тому на даному етапі потрібно було діставати інформацію з масивів, які не є постійними, 
в залежності від зміни місцязнаходження.  

Виконання тривало 5 днів.
